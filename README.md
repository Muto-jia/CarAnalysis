# 🚗 CarAnalysis — Hadoop 汽车销售数据分析项目

> 基于 Hadoop MapReduce 实现的汽车销售多维度统计分析课程项目

---

## 📌 项目简介

本项目使用 **Hadoop MapReduce** 分布式计算框架，对汽车销售数据集进行多维度统计分析，覆盖车辆属性、时空分布、用户画像三大类业务场景，共包含 9 个独立分析任务，完整展示了从数据读取、分布式计算到结果输出的全流程实现。

## 📊 项目任务列表

| 序号 | 任务名称 | 说明 |
|------|----------|------|
| 1 | 乘用/商用车辆数量与销售额分布统计 | 统计两类车辆的销量和销售额差异 |
| 2 | 年度各月份汽车销量占比统计 | 分析全年销量季节性波动规律 |
| 3 | 单月份各市区县汽车销量统计 | 分析不同区域的消费差异 |
| 4 | 购车用户男女比例统计 | 用户画像分析：性别结构 |
| 5 | 不同所有权、型号汽车销量统计 | 从所有权和车型维度分析销量分布 |
| 6 | 不同车型的用户年龄和性别分布统计 | 车型与用户画像的关联分析 |
| 7 | 不同车型销售数据统计 | 轿车、SUV、MPV 等车型整体销量对比 |
| 8 | 不同品牌汽车的发动机型号与燃料种类统计 | 品牌动力配置与燃料结构分析 |
| 9 | 同排量不同品牌汽车销量统计 | 相同排量下不同品牌的市场表现对比 |
